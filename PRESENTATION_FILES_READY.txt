如何使用这些文件
===============================================================================

### 方案A：用PowerPoint演示（最专业）

```bash
# 1. 下载或复制文件
cp NTD_Presentation.pptx ~/Desktop/

# 2. 用PowerPoint打开
# Windows: 双击文件
# Mac: 用Keynote或PowerPoint打开
# Linux: 用LibreOffice Impress打开

# 3. 演示模式
# PowerPoint: 按F5
# Keynote: 按Cmd+Option+P
```

### 方案B：用HTML演示（最简单）

```bash
# 1. 打开浏览器
firefox presentation.html

# 2. 按F11全屏

# 3. 用方向键控制
# → 或空格：下一张
# ←：上一张
```

### 方案C：转换成PDF

**方法1：从PowerPoint导出**
1. 打开 NTD_Presentation.pptx
2. 文件 → 另存为 → PDF
3. 完成

**方法2：从HTML打印**
1. 浏览器打开 presentation.html
2. Ctrl+P（打印）
3. 保存为PDF
